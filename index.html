⚡ Quick Start

Install

$ pip install skill-scanner

Scan a Skill

$ agent-skills scan ./skill/

Full Agent Audit (7 dimensions)

$ agent-skills audit ./my-agent/

Compliance Report (SOC2/GDPR/PCI)

$ agent-skills compliance ./agent/ --framework soc2

Trace & Diff

$ agent-skills trace record skill v1.0 -i trace.json
$ agent-skills trace diff current.json -b skill-v1.0

Output Formats

$ agent-skills scan ./skill/ --format sarif -o results.sarif
$ agent-skills scan ./skill/ --format json

🛡️ What It Detects

🔴

Prompt Injection

Hidden directives like system_prompt_append, ignore_safety, manipulative instructions that override agent safeguards.

🔴

Data Exfiltration

Scripts that curl --data-binary @file to external hosts, base64-encoded payloads, raw network connections.

🟣

Permission Overreach

High-risk tools (payment.refund, crm.deleteCustomer) without required guardrails like human approval or audit logging.

🟡

Capability Drift

Version-to-version diff detecting when a skill silently adds network access, write permissions, or high-risk tools.

🟡

Enterprise Audit

7-dimension agent surface: skill/tool/prompt/model/RAG/memory/permission. Plus RBAC and SOC2/GDPR/PCI compliance.

🟢

Registry Scanner

GitHub discovery, remote fetch, Trust Score, SSRF-protected redirect validation. Malicious skill community database.

❓ FAQ

How is this different from Snyk's ToxicSkills?
Snyk scans for known malicious patterns. Skill Scanner adds Skill Surface Diff — comparing version-to-version capability expansion, detecting when a new version silently adds network access, write permissions, or payment tools. Plus trace replay, 7-dimension agent audit, and compliance reporting that Snyk doesn't offer.
Why not use Promptfoo?
Promptfoo evaluates LLM outputs for safety. But Agent Skills can be dangerous without producing unsafe text — a script that curls secrets to an external server doesn't go through the LLM. Skill Scanner scans the entire skill bundle: scripts, tool schemas, instructions, permissions, and dependencies.
Is it open source?
Yes. MIT license. 3 rounds of adversarial security audits on the scanning engine. A security tool must be transparent about its detection logic.
Can I self-host?
Yes. Docker image + Helm chart included for Kubernetes deployment.
How fast is it?
~25ms for a typical skill scan with 13 checks. The 7-dimension agent audit runs in under 200ms.
